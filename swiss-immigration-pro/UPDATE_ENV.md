# Update .env.local with xAI API Key

## Add this line to your `.env.local` file:

```
XAI_API_KEY=your_xai_api_key_here
```

## Steps:

1. Open `swiss-immigration-pro/.env.local` in your editor
2. Add or update the line:
   ```
   XAI_API_KEY=your_xai_api_key_here
   ```
3. Save the file
4. Restart your dev server:
   ```bash
   npm run dev
   ```
